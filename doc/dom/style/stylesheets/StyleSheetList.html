Packagedom.style.stylesheets
Classpublic class StyleSheetList
InheritanceStyleSheetList Object

Introduced in: DOM 2 Core 

The StyleSheetList interface provides the abstraction of an ordered collection of style sheets.

The items in the StyleSheetList are accessible via an integral index, starting from 0.

DOM 2 Style item()method
public function item(index:Number):StyleSheet

Introduced in: DOM 2 Core 

Used to retrieve a style sheet by ordinal index. If index is greater than or equal to the number of style sheets in the list, this returns null.

Parameters

index:Number — Index into the collection

Returns
StyleSheet — The style sheet at the index position in the StyleSheetList, or null if that is not a valid index.
